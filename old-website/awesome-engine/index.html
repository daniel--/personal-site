Awesome Engine

The Awesome Engine is a game engine created for rapid prototyping in Game Jams. It is written in python and uses the SDL media library and the Box2D physics engine. It implements a version of an Entity-Component-Sytem architecture. While the Awesome Engine functions and has been used in several game jams, it is very much a work in progress. Every game jam introduces some new quick hacks to get the game working by the deadline and prep work for the next game jam smooths out and integrates those hacks. After several years it is starting to approach stability.

Get It

You can find the latest source code for the Awesome Engine on github here.

First install your distributions SDL dev package, then sdl2-cffi and sdl2hl. Finally install Awesome Engine.
