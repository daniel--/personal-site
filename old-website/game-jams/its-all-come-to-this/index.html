So it Has Come to This

About

So it Has Come to This was created at TOJam 10 in 2015. So it Has Come to This is a post apocalyptic survivor game in the style worker placement board games. You must send each of your survivors to a different location in order to collect resources and try try to survive. Each day there is a new random event that will affect your camp. See how long you can survive before you run out of resources.

Play It

You can find the latest source code for So it Has Come to This on github here, and an archive here.

To play on Windows first install 32bit python 2.7, then pygame, run run.py

To play on Linux install pygame from your package manager, then run run.py.

Postmortem

So it Has Come to This is incomplete. The three-day game jam proved to be too much for a two person team to implement all the required features, especially u.i. features required for this style of game in a relatively new engine. Although we got almost all the basic functionality complete we didn't have enough time to create almost any content, work on any balance, or add any polish. One interesting feature we were able to add is the selection of random names for U.S. census data. The main lesson to be learned is that working in new engines is tricky and games that require a lot of content should not be attempted with a small team at a three-day jam.
